Bienvenue sur la page Github de MyGUi.

MyGui est une librairie simple d'utilisation qui vous permet de créer votre gui SFML en 5 minutes ! Un simple fichier de configuration permet à MyGui de mettre en place des interfaces complexe et pleinement opérationnelle. Son système d'ECS en fait un outil adapté à toutes les situations.

Tutorial

Step 1

Instanciez le render système et la Factory.

SRender   render;
Factory   builder(&render);

Step 2

Créez vos objets !

Entity *cercle = builder.buildCircle();
cercle->getAttr<GEOMETRY>()->set("x", 100)->set("y", 100)->set("w", 10);
cercle->getAttr<COLOR>()->setRGBA(255, 255, 0, 100);
cercle->getAttr<EVENT>()->setEvent(sf::Event::MouseButtonPressed, &mouse_up_fct);

Step 3

La boucle d'event !

while(!terminer)
  {
    sf::Event e;
    while (render.update(e))
  {
    if (e.type == sf::Event::Closed)
      terminer = true;
    cercle->getAttr<EVENT>()->checkEvent(e);
  }
render.clear();

cercle->getAttr<DRAW>()->draw();

render.display();

}
